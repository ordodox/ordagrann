# Ordagrann

Ett dagligt svenskt ordpussel. Nio bokstäver, en ny uppsättning varje dag —
hitta så många ord du kan som använder den gula mittenbokstaven.

## Så spelar du

Öppna sidan (eller `index.html` från disk — den fungerar utan internet).
Klicka på bokstäverna eller skriv på tangentbordet för att bygga ord:

- Varje ord måste vara minst tre bokstäver långt.
- Varje ord måste innehålla den gula mittenbokstaven.
- Varje bokstav får bara användas så många gånger den finns i rutnätet.
- Bara ord ur Svenska Akademiens ordlista räknas.

**Enter** skickar ordet, **Backspace** raderar sista bokstaven, **Delete**
rensar, och **Mellanslag** blandar de yttre bokstäverna. Slå på
9-bokstavsläget för den ultimata utmaningen: bara ord som använder alla nio.

## Varje dag, samma pussel

Samma datum ger samma bokstäver på alla enheter — dela din dag med en länk
(`?date=…`), jämför i gruppchatten, och samla prestationer allt eftersom.
Gårdagens och äldre pussel finns kvar; bläddra bakåt med pilarna.
